CURRENT PROJECT

OVERVIEW

Repartease is a simple party game to see how depraved your friends are. It's still a work in progress, so it'll be buggy. It's built on a Node.js Express / MongoDB / React stack.

Live Game: repartease.herokuapp.com.

GAMEPLAY

Starting a Game

  1. Type your name in and click "Start Game"
  2. Copy the Game Code and give it to your friends (or enter it in the 2nd window below)
  3. Once everyone has joined, hit "All In" and you're ready to play

Joining a Game

  1. Click the "Join New Game" button
  2. Enter your name and GameCode (get that from whoever started the game)
  3. Wait for the person who started the game to press the "All In" button and get ready to play
